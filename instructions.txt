Για να λειτουργήσει ο κώδικας του αρχείου test6.py πρέπει να γίνουν τα παρακάτω:

Εγκατάσταση μιας έκδοσης της python από την ιστοσελίδα https://www.python.org/
Εγκατάσταση των απαραίτητων βιβλιοθηκών. Θα πρέπει να εκτελεστούν στην γραμμή εντολών των windows οι εντολές:
pip install web3 
pip install py-solc-x
pip install python-dotenv
pip install eth-abi
pip install eth-account
Εγκατάσταση της ganache που μπορεί να βρεθεί στην ιστοσελίδα https://archive.trufflesuite.com/ganache/
Εκτέλση της ganache
Επιλογή new workspace
Στην καρτέλα που θα εμφανιστεί επιλέγουμε το κουμπί chain και στο πεδίο gas limit εισάγουμε την τιμή 10000000 (10 εκατομμύρια) και πατάμε το κουμπί start
Η ganache εμφανίζει 10 λογαριασμούς από τους οποίους χρειαζόμαστε μόνο έναν
Πηγαίνουμε τέρμα δεξιά σε έναν λογαριασμό και πατάμε στο εικονίδιο ενός κλειδιού
Αν πατηθεί το κλειδί εμφανίζεται η διεύθυνση του λογαριασμού και ένα ιδιωτικό κλειδί, το οποίο αντιγράφουμε.
Στον φάκελο με όλα τα αρχεία πηγαίνουμε στο αρχείο key.env και κάνουμε επικόλληση στο κλειδί που μόλις αντιγράψαμε
Αφού ολοκληρωθούν όλα τα παραπάνω πατάμε δεξί κλικ με το ποντίκι στο αρχείο test6.py, από τις επιλογές που εμφανίζονται, πατάμε στην επιλογή Edit with IDLE.
Αφού γίνει αυτό ανοίγει ο κώδικας με την python και μπορούμε να πατήσουμε το κλειδί πληκτρολογίου F5 για να τρέξει ο κώδικας.

Για να λειτουργήσει ο κώδικας του αρχείου test6.exe το μόνο βήμα που χρειάζεται να εκτελεστεί είναι η εγκατάσταση της ganache και η αντιγραφή 
ενός ιδιωτικού κλειδιού, το οποίο θα αντιγραφεί στο αρχείο key.env που βρίσκεται μέσα στον φάκελο dist. Μετά μπορούμε να τρέξουμε το εκτελέσιμο
αρχείο και ο κώδικας θα λειτουργήσει κανονικά.
